# Engramia

**Reusable execution memory for AI agents.**
Agents don't learn from previous runs — Engramia fixes that.

Core loop: Learn → Recall → Compose → Evaluate → Improve

## Projects

- 📦 [engramia](https://github.com/engramia/engramia) — Python library, REST API, MCP server
- 🖥️ [dashboard](https://github.com/engramia/dashboard) — Web UI for managing memory
